# Changelog

## 1.0.0
- Initial release
- Tracks jumps during ENCOUNTER_START/ENCOUNTER_END encounters
- Silent addon comms for real-time updates during encounters
- Small translucent on-screen Top 5 leaderboard
- Class-colored names (fallback to white)
- Timeout + fade-out for stale/non-reporting players
- End-of-encounter leaderboard posted to chat by the winner only
- Basic slash commands for show/hide, lock/unlock, scale, timeout, fade
